# Contributing to DocSnap

DocSnap is a Chrome extension built to support technical documentation writers and engineers. Contributions must prioritise clarity, speed, and practical usability in real documentation workflows.

---

## Scope of Contributions

Contributions are accepted in the following areas:

- Documentation capture and annotation workflows
- UI clarity and usability improvements
- Performance optimisation and stability fixes
- Code maintainability and developer experience
- Bug fixes and edge case handling

Do not introduce features that do not directly improve documentation efficiency.
